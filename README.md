About the Project

This project was developed using Python, HTML, CSS, JavaScript, Oracle SQL, and Power BI technologies.

This project is a part of an internship of Data Analytics and Generative AI of ABB Tech Academy in Baku, in Azerbaijan.

Database: Oracle SQL (combining several data, storing, and loading to Power BI)

Cleaning and Visualization: Power BI
